### 1. Тип boolean 

Принимает два значения true \ false.
Пример
```Java
int age = 70;  
boolean isSenior = (age > 65);  
if (isSenior)  
    System.out.println("Пора на пенсию");

Эквивалент 

int age = 70; 
if (age > 65) 
	System.out.println("Пора на пенсию");
```
